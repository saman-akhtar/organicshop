# Organic Shop

An Angular e-commerce web application for an organic products store. Deployed on Firebase Hosting.

## Features

- Product catalog with categories
- Shopping cart functionality
- Firebase backend integration
- Responsive UI built with Angular Material / Bootstrap

## Tech Stack

- **Angular 6** — component-based frontend framework
- **TypeScript** — typed JavaScript
- **Firebase** — hosting and backend services
- **Angular CLI** — build, serve, and deploy tooling

## Run locally

```bash
git clone https://github.com/saman-akhtar/organicshop.git
cd organicshop
npm install
ng serve
```

Opens at [http://localhost:4200](http://localhost:4200)

## Deploy

```bash
npm run build
firebase deploy
```
